## 구현내용
1. 입력기능
   1. 자동차 이름 입력받기
   2. 시도할 횟수 입력받기
2. 레이싱 경주 준비 (=초기 Car 세팅하기)
   1. name은 "," 기준으로 split 하고 map에 저장
   2. position은 0으로 초기화
3. 레이싱 경주 시작 (=시도할 횟수만큼 반복)
   1. 0~9 사이의 랜덤값 생성
   2. 생성된 값이 4이상이면 전진, 아니면 멈춤
   3. 전진한 자동차의 position을 1 증가

## 기능 목록 
- [x] 자동차는 이름을 가진다.
- [x] 자동차의 이름은 5자를 초과할 수 없음
- [x] 자동차는 4 이상일 경우 움직인다
- [x] 자동차는 4 미만인 경우 멈춰있는다
