## Step2 : 문자열 계산기
### 요구사항
1. 사칙연산의 계산우선순위가 아닌 입력값에 따라 계산순서가 결정된다
   (예 : 2 + 3 * 4 / 2 = 10)
2. 메서드를 최대한 분리시킨다.

  * 사칙연산을 모두 포함 하는 경우

* [x] 예외처리 테스트 코드 작성
